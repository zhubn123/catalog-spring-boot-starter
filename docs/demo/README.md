# Demo Assets

这个目录用于存放 README 演示截图。

## 推荐文件名

- `01-tree-overview.png`
- `02-binding-and-query.png`
- `03-business-example.png`
- `04-debug-logs.png`

## 推荐截图范围

### 01-tree-overview.png
- 左侧目录树
- 当前节点概览
- 能看出拖拽排序入口

### 02-binding-and-query.png
- 叶子节点绑定区域
- 路径查询或局部树查询结果

### 03-business-example.png
- 业务示例区域
- 目录挂载和叶子节点绑定场景

### 04-debug-logs.png
- 前端 API 日志页签
- 同一屏里尽量带上终端中的请求日志或 SQL 日志

## 建议

- 优先用浅色背景截图，GitHub README 里更清楚。
- 一张图只讲一个主题，不要把所有功能塞进同一张图。
- 第一版先保证“能看懂项目做什么”，不必追求大而全。
